Introducción

La API de RutVans proporciona servicios para la gestión de empresas de transporte, incluyendo administración de usuarios, empresas, quejas y métricas del sistema. La API está construida sobre Laravel y sirve como backend para la aplicación móvil Flutter.

Autenticación

Nota: Todas las rutas protegidas requieren un token Bearer en el header de la petición: Authorization: Bearer {token}

Obtener Token

POST /api/auth/login 
{
    "email": "usuario@ejemplo.com",
    "password": "contraseña"
}

Endpoints

Autenticación y Usuarios

POST /api/auth/mobile-login - Iniciar sesión móvil
POST /api/auth/mobile-register - Registrar nuevo usuario móvil 
// Solicitud
{
    "name": "Usuario Ejemplo",
    "email": "usuario@ejemplo.com",
    "password": "contraseña"
}

// Respuesta (201 Created)
{
    "success": true,
    "message": "Usuario registrado correctamente. Por favor, verifica tu correo electrónico antes de iniciar sesión."
}
GET /api/auth/user - Obtener usuario actual
POST /api/auth/logout - Cerrar sesión

Perfil de Usuario

GET /api/profile - Obtener perfil
PUT /api/profile - Actualizar perfil
POST /api/profile/photo - Actualizar foto de perfil
PUT /api/profile/password - Cambiar contraseña

Quejas y Comentarios

GET /api/complaints - Listar quejas (público)
POST /api/complaints - Crear queja
GET /api/complaints/user/{userId} - Quejas por usuario
POST /api/comments - Crear comentario
GET /api/comments - Listar comentarios

Formularios y FAQs

POST /api/forms - Enviar formulario
GET /api/forms - Listar formularios
GET /api/forms/user - Formularios por usuario
GET /api/faqs - Listar FAQs

Empresas (Companies)

GET /api/super-admin/companies - Listar todas las empresas
GET /api/super-admin/companies/{id} - Obtener empresa específica
GET /api/super-admin/companies/stats - Estadísticas de empresas

Admin Dashboard

GET /api/super-admin/admin/stats - Estadísticas generales
GET /api/super-admin/admin/forms-monthly - Formularios mensuales
GET /api/super-admin/admin/sites-count - Conteo de terminales
GET /api/super-admin/admin/units-count - Conteo de unidades
GET /api/super-admin/admin/dashboard - Datos del dashboard
GET /api/super-admin/complaints/monthly - Quejas mensuales

Admin Gestión

GET /api/super-admin/complaints - Listar todas las quejas
PUT /api/super-admin/complaints/{id} - Responder/actualizar queja
DELETE /api/super-admin/complaints/{id} - Eliminar queja

Servicios Flutter

Servicio Descripción Métodos Principales
AdminService Gestiona funcionalidades administrativas
  • fetchUsersCount()
  • fetchSitesCount()
  • fetchEmpresasCount()
  • fetchMonthlyComplaints()
  • fetchForms()
CompaniesService Gestiona operaciones relacionadas con empresas
  • fetchCompanies()
  • fetchCompanyById(String id)
AuthService Maneja autenticación y gestión de usuarios
  • login(String email, String password)
  • register(String name, String email, String password) - Devuelve mensaje de verificación de correo
  • fetchCurrentUser(String token)
  • logout()
ProfileService Gestiona el perfil del usuario
  • editProfile(String name, String email)
  • updatePassword(String currentPassword, String newPassword)
  • uploadProfilePhoto(File imageFile)
ComplaintsService Gestiona quejas y sugerencias
  • submitComplaint(String content, String userId)
  • fetchComplaintsByUser(String userId)
  • respondComplaint(String id, String response)
  • deleteComplaint(String id)
CommentsService Gestiona comentarios públicos
  • fetchComentarios()
  • submitComment(String content, String userId)
FaqsService Gestiona preguntas frecuentes
  • fetchFaqs()
  • createFaq(String question, String answer)
  • updateFaq(String id, String question, String answer)
  • deleteFaq(String id)
FormsService Gestiona formularios y cotizaciones
  • submitForm(Map<String, dynamic> formData)
  • fetchUserForms(String userId)
  • fetchAllForms()

Controladores Laravel

Controlador Descripción Rutas Asociadas
AuthController Maneja autenticación y registro /api/auth/*
CompanyController Gestión de empresas /api/super-admin/companies/*
AdminController Funciones administrativas /api/super-admin/admin/*
