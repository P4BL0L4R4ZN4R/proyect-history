API - Perfil Chofer

Endpoints para gestión de operaciones de conductores

https://rutvans.com/api/driver

Introducción

La API Driver de RUTVANS permite a los conductores interactuar con el sistema para gestionar sus operaciones diarias.

Características principales

  • Iniciar sesión - Autenticación segura
  • Cerrar sesión - Terminar sesiones
  • Gestión de perfil - Actualización de datos
  • Rutas y servicios - Consulta de horarios
  • Boletos y envíos - Gestión de entregas
  • Incidencias - Reporte de problemas
  • Notificaciones - Alertas importantes
  • Estadísticas - Métricas de desempeño

Endpoints

POST /login
Permite a los conductores iniciar sesión en el sistema.
Ejemplo de solicitud
POST https://rutvans.com/api/driver/login
Parámetro Tipo Requerido Descripción
email String Sí Correo electrónico
password String Sí Contraseña
Código Descripción
200 Autenticación exitosa
400 Datos inválidos
401 Credenciales incorrectas
GET /incident/{id}
Obtiene los detalles de una incidencia específica reportada por el chofer.
Ejemplo de solicitud
GET https://rutvans.com/api/driver/incident/1
Ejemplo de respuesta
{ "success": true, "data": [ { "id": 1, "details": "hay algo", "user_id": 1, "category": "Paso algo en carretera", "image": null, "urgency": "Alta muy alta", "updated_at": "2025-08-13 16:48:48", "created_at": "2025-08-13 16:48:51" } ] }
Parámetro Tipo Descripción
id Integer ID de la incidencia a consultar
Código Descripción
200 OK - Incidencia obtenida correctamente
404 Not Found - Incidencia no encontrada
DELETE /incident/delete/{id}
Elimina una incidencia reportada por el chofer. Solo disponible para incidencias en estado "pendiente".
Código Descripción
200 OK - Incidencia eliminada correctamente
403 Forbidden - No tiene permisos para eliminar
404 Not Found - Incidencia no encontrada
GET /user/edit/{id}
Obtiene los datos actuales del perfil del chofer para su edición.
Ejemplo de solicitud
GET https://rutvans.com/api/driver/user/edit/1
Ejemplo de respuesta
{ "success": true, "data": { "id": 1, "name": "Carlos Arjona", "email": "carlos@gmail.com", "phone": "+52 55-1234-5678" } }
PUT /user/update/{id}
Actualiza la información del perfil del chofer. Se pueden enviar los parámetros como query parameters o en el cuerpo de la solicitud en formato JSON.
Ejemplo de solicitud
PUT https://rutvans.com/api/driver/user/update/1?name=Carlos%20Vilarta%20M
Headers requeridos
| Header | Valor | |---------------------|-----------------------| | Authorization | Bearer [token] | | Content-Type | application/json |
Parámetros aceptados
Parámetro Tipo Requerido Descripción
name String Opcional Nombre completo del chofer
email String Opcional Correo electrónico
phone String Opcional Teléfono en formato internacional
Ejemplo de respuesta exitosa
{ "success": true, "message": "Perfil actualizado correctamente", "data": { "id": 1, "name": "Carlos Vilarta M", "email": "carlos@gmail.com", "phone": "+52 55-1234-5678" } }
Posibles códigos de respuesta
Código Descripción
200 OK - Actualización exitosa
400 Bad Request - Datos inválidos o faltantes
401 Unauthorized - Token inválido o faltante
404 Not Found - Usuario no encontrado

driverId Integer ID del conductor
GET /route-unit-schedules
Obtiene el listado completo de horarios de rutas asignadas al chofer.
Ejemplo de solicitud
GET https://rutvans.com/api/driver/route-unit-schedules
Ejemplo de respuesta
{ "success": true, "data": [ { "id": 28, "route_unit_id": 4, "schedule_date": "2025-08-13", "schedule_time": "00:00:45", "status": "Activo", "created_at": "2025-08-13T19:06:33.000000Z", "updated_at": "2025-08-13T19:06:33.000000Z", "route_unit": null } ] }
GET /route-unit-schedules/{id}
Obtiene los detalles específicos de un horario de ruta particular.
Ejemplo de solicitud
GET https://rutvans.com/api/driver/route-unit-schedules/1
Ejemplo de respuesta
{ "success": true, "data": { "id": 28, "route_unit_id": 4, "schedule_date": "2025-08-13", "schedule_time": "00:00:45", "status": "Activo", "created_at": "2025-08-13T19:06:33.000000Z", "updated_at": "2025-08-13T19:06:33.000000Z" } }
Descripción de campos
Campo Tipo Descripción
schedule_date Date Fecha programada para la ruta (YYYY-MM-DD)
schedule_time Time Hora de inicio programada (HH:MM:SS)
status String Estado actual (Activo, Completado, Cancelado)
GET /freights
Obtiene el listado de todos los fletes asignados al chofer.
Ejemplo de respuesta
{ "flete": [ { "id": 1, "folio": "002", "service_id": 1, "start_date": "2025-08-16", "end_date": "2025-08-16", "start_time": "10:30:00", "end_time": "16:30:00", "status": "pendiente", "amount": "350.00", "name": "Marcos", "origin": "Merida", "driver_id": 1, "destination": "Kopoma", "number_people": 14, "created_at": "2025-08-13 19:01:44", "updated_at": "2025-08-14 22:10:57" } ] }
GET /freights/{id}
Obtiene los detalles completos de un flete específico.
Ejemplo de solicitud
GET https://rutvans.com/api/driver/freights/1
PUT /freights/update/{id}
Actualiza el estado básico de un flete (ej: pendiente → en_progreso).
Ejemplo de solicitud
PUT https://rutvans.com/api/driver/freights/update/1?status=en_progreso
Ejemplo de respuesta
{ "message": "Flete updated successfully" }
Parámetro Tipo Requerido Descripción
status String Sí Nuevo estado del flete
PUT /freights/edit/{id}
Modifica detalles específicos del flete (fechas, horarios, detalles).
Ejemplo de solicitud
PUT https://rutvans.com/api/driver/freights/edit/1?folio=003
Parámetro Tipo Requerido Descripción
folio String No Nuevo folio del flete
start_date Date No Nueva fecha de inicio (YYYY-MM-DD)
start_time Time No Nueva hora de inicio (HH:MM:SS)
GET /shipments
Obtiene el listado de todas las entregas asignadas al chofer.
Ejemplo de solicitud
GET https://rutvans.com/api/driver/shipments
Ejemplo de respuesta
{ "shipments": [ { "id": 1, "folio": "002", "route_unit_id": 4, "service_id": 2, "sender_name": "Eduardo Canul", "receiver_name": "Julia Noh", "package_description": "Bien, todo bien jj", "package_image": null, "status": "Pendiente", "amount": "1.00", "created_at": "2025-08-13 19:17:26", "updated_at": "2025-08-14 23:37:44", "user_id": null, "locality": "Merida" } ] }
GET /shipments/{id}
Obtiene los detalles completos de una entrega específica.
Ejemplo de solicitud
GET https://rutvans.com/api/driver/shipments/1
Ejemplo de respuesta
{ "shipments": { "id": 1, "folio": "002", "route_unit_id": 4, "sender_name": "Eduardo Canul", "receiver_name": "Julia Noh", "status": "Pendiente", "locality": "Merida" } }
PUT /shipments/update/{id}
Actualiza el estado o información de una entrega específica.
Ejemplo de solicitud
PUT https://rutvans.com/api/driver/shipments/update/1?sender_name=Eduardo%20Canul%20Pot
Ejemplo de respuesta
{ "message": "shipments updated successfully" }
Campos actualizables
  • status: Cambiar estado (Pendiente, En camino, Entregado, etc.)
  • sender_name: Nombre del remitente
  • receiver_name: Nombre del destinatario
  • package_description: Descripción del paquete
