Rutvans Móvil - API Perfil Clientes

Introducción

Esta documentación describe las rutas disponibles en la API de Rutvans Móvil para el perfil de clientes, incluyendo parámetros, ejemplos de solicitud y respuesta.

URL Base

https://rutvans.com/api

Autenticación

La mayoría de los endpoints requieren autenticación mediante token JWT. Incluya el token en el encabezado de la solicitud:

Authorization: Bearer {token}

Para obtener un token, utilice el endpoint /login.

Métodos POST

POST /shipment

Crea un nuevo envío o paquetería.

URL Completa

https://rutvans.com/api/shipment

Parámetros (body)

Nombre Tipo Descripción
origin string Ciudad de origen (requerido)
destination string Ciudad de destino (requerido)
description string Descripción del paquete
weight number Peso en kg (requerido)
dimensions string Dimensiones (LxAxH en cm)

Respuesta exitosa

{
    "success": true,
    "message": "Envío creado exitosamente",
    "shipment_id": 1,
    "tracking_number": "RTV-001"
}

Códigos de estado HTTP

Código Descripción
200 OK Solicitud exitosa
201 Created Recurso creado exitosamente
400 Bad Request Solicitud incorrecta (faltan parámetros o son inválidos)
401 Unauthorized No autenticado (token inválido o no proporcionado)
403 Forbidden No tiene permisos para realizar esta acción
404 Not Found Recurso no encontrado
422 Unprocessable Entity Error de validación en los datos enviados
500 Internal Server Error Error interno del servidor
